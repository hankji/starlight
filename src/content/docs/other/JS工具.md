## JS 框架 
* [Vue](http://cn.vuejs.org/guide/)

## JS UI 工具集
* 选项卡功能 (Tabs)
* 提示工具条功能 (ToolTips)
* 信息滚动功能 (Scrollable)
* 遮罩效果 (Overlay)
* 突出效果 (Expose)
* Flash 嵌入
* 图片突出, 表单内的浮层
* 验证
* 滚动条取值
* 日期输入
* jQuery UI
